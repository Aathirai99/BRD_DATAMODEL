# BRD to Data Model Generator

Convert Business Requirements Documents into Informatica MDM data models using AI.

## Quick Start
```bash
# 1. Install dependencies
pip install pandas openpyxl

# 2. Place your BRD Excel file in the project folder

# 3. Run the full pipeline
python run_full_pipeline.py

# 4. Follow instructions to use Cursor AI
#    (Cursor will generate JSON, then run step 3 again for visualizations)
```

## Usage

### Automatic (Recommended)
```bash
# Auto-detect Excel file and run full pipeline
python run_full_pipeline.py
```

### Specify BRD File
```bash
# Use specific Excel file
python run_full_pipeline.py --brd "path/to/your/brd.xlsx"
```

### Generate Visualizations Only
```bash
# After Cursor generates JSON, generate visualizations
python run_full_pipeline.py --visuals-only
```

## Workflow

1. **Place Excel BRD** in project folder
2. **Run pipeline**: `python run_full_pipeline.py`
3. **Cursor AI**: Process the generated prompt (manual step)
4. **Auto-generate**: Visualizations created automatically when JSON exists

## Outputs

- **Prompt file** (`outputs/*_prompt.txt`) - For Cursor AI
- **JSON data model** (`outputs/*_response.json`) - Generated by Cursor
- **Draw.io diagram** (`outputs/*_data_model.drawio`) - Visual data model
- **HTML report** (`outputs/*_data_model_report.html`) - Interactive report with traceability
